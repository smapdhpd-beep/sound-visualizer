# 06-预设格式规范

## 配色预设（Color Theme）

```json
{
  "id": "warmSand",
  "name": "暖沙",
  "palette": {
    "background": [10, 10, 15],
    "particle": [255, 220, 180],
    "accent": [255, 180, 100],
    "glow": [255, 200, 150],
    "trailAlpha": 0.12
  }
}
```

## 模式参数预设（Mode Preset）

```json
{
  "mode": "chladni",
  "params": {
    "particleCount": 4000,
    "viscosity": 0.88,
    "strength": 1.2,
    "symmetry": 2,
    "trailAlpha": 0.12,
    "glowStrength": 3,
    "particleSize": 1.5,
    "responseSpeed": 0.08,
    "complexity": 1,
    "boundaryShape": "circle"
  },
  "theme": "warmSand"
}
```

## 快照（Snapshot）

```json
{
  "version": "1.0",
  "createdAt": "2026-06-17T12:00:00Z",
  "mode": "chladni",
  "theme": "ink",
  "global": {
    "audioSmoothing": 0.85
  },
  "params": {
    "viscosity": 0.88,
    "symmetry": 2,
    "boundaryShape": "hexagon"
  }
}
```

## URL Hash 分享格式

```
#mode=chladni&theme=aurora&boundaryShape=square&symmetry=2
```
