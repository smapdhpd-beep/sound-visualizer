# 01-项目概述

## 项目名称
Sound Visualizer（声音可视化器）

## 定位
浏览器端 PWA，将麦克风或本地音乐转化为实时互动视觉。
核心体验：好酷、治愈、可干预、可导出个人作品。

## 目标平台
- 桌面浏览器（Chrome / Safari / Firefox / Edge）
- 移动端浏览器（iOS Safari / Android Chrome），响应式适配

## 输入输出
- **输入**：麦克风实时流 / 本地音乐文件（<input type="file" accept="audio/*">）
- **输出**：全屏 Canvas 2D / WebGL 视觉，支持截图导出 PNG

## 视觉模式（单模式聚焦）
| 模式 | 英文名 | 状态 | 描述 |
|------|--------|------|------|
| 克拉尼 | chladni | **v1.0 已封版** | 2D 驻波势场，粒子滑向波节线 |

> **历史模式**：光尘（dust）、频谱瀑布（waterfall）、液态（liquid）均已废弃归档，不再维护。

## 六种边界形态
克拉尼模式支持以下边界条件，通过调参面板「边界形态」切换：

| 形态 | 数学基础 | 视觉特征 |
|------|----------|----------|
| 圆形 | 极坐标驻波 `sin(m·π·r)·cos(n·θ)` | 同心圆环与径向节线 |
| 方形 | 笛卡尔驻波 `sin(m·π·x)·sin(n·π·y)` | 网格状节线，镜像对称 |
| 环形 | 归一化径向坐标 + 内孔 0.28 | 同心圆环带，中孔为空 |
| 椭圆 | 变形极坐标 + 二阶猫眼调制 | 挤压变形，猫眼状图案 |
| 三角形 | 重心坐标 + 三边距离乘积 | 平行于三边的直线网格 |
| 六边形 | 三对平行边距离乘积 | 蜂窝状六边形网格 |

## 配色与导出
- **5 套全局配色**：暖沙、霓虹、水墨、极光、幻彩（audioDriven 色相漫游）
- **截图导出**：点击 📷 即可保存当前 Canvas 为 PNG（纯净画面，不含 UI）

## 音乐优化方向
- **电子乐**：低频 Bass / Onset 节拍检测驱动视觉爆发
- **人声**：中频纹理 / 频谱质心驱动细腻呼吸与闪烁

## 快速开始
1. 打开 `index.html`（需本地服务器，建议 `npx serve` 或 VSCode Live Server）
2. 点击「播放本地音乐」或「使用麦克风」
3. 右下角 ⚙️ 展开调参面板，可切换边界形态与微调参数
4. 鼠标/触摸划过画面与粒子互动
