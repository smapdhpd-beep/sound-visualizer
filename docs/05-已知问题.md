# 05-已知问题

## 当前版本（v1.0 封版）

1. **Chladni 模态跳变散开**
   - Onset 触发 m/n 随机化时，粒子瞬间失去旧波节约束，视觉上会先炸开再聚拢。
   - 设计固有行为；可后续加入「模态过渡时长」参数平滑插值。

2. **iOS 底部安全区**
   - Tweakpane 底部抽屉已适配 `env(safe-area-inset-bottom)`，但小屏机型仍可能遮挡。

3. **移动端待测**
   - iOS Safari / Android Chrome 真机尚未全面测试。
   - 待验证项：4000 粒子下 FPS 是否稳定、触摸交互是否流畅、DPR 缩放是否正常、音频启动是否需要用户手势唤醒。

4. **截图不包含 UI**
   - 截图功能仅合成 Canvas 层，不捕获顶部栏或 Tweakpane。
   - 这是设计意图（纯净作品导出）。
